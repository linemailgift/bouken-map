# 冒険プロジェクト 企画提案書（おさるさん向け）

## 背景：ゆたぼん号とは

ゆたぼん（ゆたぼんスタディー号）は、不登校の子供たちに勇気を与えるために日本全国を回った。
- キャッチフレーズ：**「人生は冒険だ」**
- 「学校に行きたければ行けばいい。行きたくなければ行かなくてもいい」という新しい世界観
- 当時バッシングされたが、今では人気者に
- 子供たち、そして子供を持つ親に勇気を与えた存在

## 企画概要

ちいくじがゆたぼん号を引き継ぎ、沖縄から出発して全国47都道府県を旅する。

## ちいくじが全国でやる3つのこと

### 1. 工作ワークショップ（子供向け）
- 全国の子供たちに工作をやりに会いに行く
- 学校や会場を借りてワークショップを開催
- 工作の認知度を上げ、ちいくじの露出度を高める

### 2. SNSマーケティング出張コンサル（大人向け）
- 会社を辞めたい人、副業したい人、学校辞めたい職員など
- SNSマーケティングで起業・副業ができることを現地で対面で伝える
- パソコンを触りながら一対一でお話しする出張コンサル

### 3. おさる講座生との対談（発信）
- 全国各地のおさる講座生に直接会いに行く
- 対談して成功事例を発信
- 「会社辞めたいけど迷っている人たち」の勇気になるコンテンツ

- おさる講座生との対談コンテンツが、新規の集客に貢献
- わずかながらでもおさるさんの集客の力になれる
- コミュニティの絆をリアルに深める機会
